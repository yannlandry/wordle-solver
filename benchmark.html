Wordle Solver

Benchmark

This tool runs the Wordle Solver against all words in either the answer list (nearly 3000) or all words in the accepted guesses list (13000). Note that the Wordle Solver is slightly optimized to win against the list of answers.


Note: The benchmark tool will run in your browser and is not very well optimized. Expect your window to become unresponsive for up to several minutes once the test is launched. For better visibility, the JavaScript console occasionally displays progress messages.
